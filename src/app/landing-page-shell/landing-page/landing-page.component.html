How It Works

Fun fact - it helped build this landing page

This is a simple frontend only app which parses Tailwind UI code and converts it into Angular component code as best as it can -- all in the browser.

  • What this is

    This is a little tool created by a lover of Tailwind CSS and Tailwind UI to help speed up his Angular development. You may find it useful too.

  • What this is not

    This tool is not associated with the creators of Tailwind UI and does not produces already made Angular versions of Tailwind UI designs. To use this tool, you need to have your own access to a Tailwind UI account.
